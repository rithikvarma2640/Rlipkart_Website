# Rlipkart E-commerce Website

A modern e-commerce platform inspired by Flipkart's design principles, featuring an AI-powered shopping assistant.

## Features

- **Product Catalog**: Browse products across multiple categories including Electronics, Clothing, Footwear, Home & Kitchen, Sports, and Accessories
- **Smart Search**: Real-time search functionality to find products by name, description, or category
- **Category Filtering**: Quick category navigation with a dedicated category bar
- **Shopping Cart**: Full cart management with quantity updates and item removal
- **AI Chatbot**: Intelligent shopping assistant that helps you:
  - Search for products
  - Get personalized recommendations
  - Find affordable options
  - Answer product queries
  - Guide through the shopping experience
- **Responsive Design**: Flipkart-inspired UI with blue and orange color scheme, clean typography using Inter font

## Design Inspiration

The design follows Flipkart's established brand guidelines:
- **Colors**: Primary Blue (#1F74BA), Yellow (#F7E200), Orange (#F09120)
- **Typography**: Inter font family for optimal readability
- **Layout**: Clean, hierarchical design with clear visual separation
- **UX Patterns**: Sticky header, category navigation, grid-based product layout

## Technology Stack

- **Frontend**: React with Vite
- **Backend**: Supabase (PostgreSQL database)
- **Styling**: Inline CSS-in-JS for component-scoped styles
- **AI Assistant**: Rule-based chatbot with product recommendation engine

## Getting Started

The application is already configured and ready to use. The database is pre-populated with sample products.
